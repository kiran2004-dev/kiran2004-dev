### 👋 Welcome to my GitHub!

I'm **Kiran Gollapalli**, an Embedded Systems Developer at **Sana Tech Inc.**

- 🔭 Currently building **[TankMate](https://github.com/kiran2004-dev/tankmate)** — a solar-powered IoT tank monitor
- 📡 Working with **ESP32, LoRa 433MHz, BLE (NimBLE), ESP-NOW**
- 📱 Building mobile apps with **Flutter + Firebase**
- 🤖 Integrating **AI APIs** into embedded hardware
- ⚡ Fun fact: My firmware handles 3 radios simultaneously — BLE, LoRa, and ESP-NOW
